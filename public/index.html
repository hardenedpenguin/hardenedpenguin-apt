Hardened Penguin APT Repository

Debian packages for hardenedpenguin projects (AllStar Link nodes and related tools).

Quick start

Install the archive keyring package once, then use apt normally:

curl -fsSLO https://hardenedpenguin.github.io/hardenedpenguin-apt/pool/main/h/hardenedpenguin-archive-keyring/hardenedpenguin-archive-keyring_1.0_all.deb
sudo apt install ./hardenedpenguin-archive-keyring_1.0_all.deb
sudo apt update
sudo apt install skywarnplus-ng-all supermon-ng

Full instructions: GitHub repository.
